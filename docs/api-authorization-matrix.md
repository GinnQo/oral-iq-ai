# API Authorization Matrix

Classification values are restricted to: PUBLIC, AUTHENTICATED, AUTHENTICATED_PLUS_SUBSCRIPTION, TEACHER_ONLY, SCHOOL_ADMIN_ONLY, SUPER_ADMIN_ONLY, INTERNAL_SERVICE_ONLY, WEBHOOK_SIGNATURE_VERIFIED.

| Route path | Methods | Classification | Authentication requirement | Subscription requirement | Allowed roles | Ownership requirement | Special verification | Reason |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| /api/auth/[...nextauth] | GET, POST | PUBLIC | None | No | Any | None | NextAuth OAuth state/callback protections | OAuth login/session lifecycle must remain publicly reachable. |
| /api/auth/google-token | GET | AUTHENTICATED_PLUS_SUBSCRIPTION | Signed-in user required | Active subscription required | STUDENT, TEACHER, SCHOOL_ADMIN, SUPER_ADMIN, ADMIN | User can only retrieve their own Google account token | Refresh token flow with reconnect-required fallback | Server-only token broker replaces browser token exposure. |
| /api/auth/sessions | GET, DELETE | AUTHENTICATED | Signed-in user required | No | STUDENT, TEACHER, SCHOOL_ADMIN, SUPER_ADMIN, ADMIN | User-scoped by authenticated userId | Scope validation (`others` or `all`) | Session listing and revocation is account-security functionality. |
| /api/auth/sessions/[id] | DELETE | AUTHENTICATED | Signed-in user required | No | STUDENT, TEACHER, SCHOOL_ADMIN, SUPER_ADMIN, ADMIN | Target session must belong to caller userId | Session ID ownership check | Prevents revoking another user session. |
| /api/subscription/status | GET | AUTHENTICATED | Signed-in user required | No | STUDENT, TEACHER, SCHOOL_ADMIN, SUPER_ADMIN, ADMIN | User-scoped billing context only | Trial bootstrap in billing resolver | Required for renewal flow even when inactive. |
